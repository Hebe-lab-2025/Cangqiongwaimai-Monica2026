### 🔤 **“Druid” 的国际音标（IPA）

**/ˈdruːɪd/**

读音拆解：

* **/druː/** 像 “德鲁——”

### 📚 **“Druid” 的含义**

**本义（最常见的英语意思）：**

* 古凯尔特文化中的 **祭司、德鲁伊**
* 角色：祭祀、智者、治疗者、占卜者
* 常出现在：凯尔特历史、奇幻小说、游戏、魔兽世界等

**引申含义：**

* 与 **自然、魔法、森林、智慧** 相关的形象

---

### 💻 在计算机领域（例如阿里巴巴 Druid）

“Druid” 被用作软件名称通常是因为：

* 象征 **守护者**
* 管理“自然之力” → 比喻 **管理数据流/连接**
* 表达 **强大而智慧的系统**

👉 例如：

* **Druid 数据库连接池（阿里）**
* **Apache Druid 分布式分析型数据库**

## ✅ 常见的数据源 / 连接池有哪些？

在 Java（尤其是 Spring / Spring Boot）里常用的数据源组件有：

### 1️⃣ HikariCP（Spring Boot 默认）

* Spring Boot 2.x 默认
* ⭐ 性能最好、最轻量
* 适合高并发生产环境

### 2️⃣ Druid（阿里巴巴 Druid）

* ⭐ 中国项目里最常见
* 功能强：监控、SQL 统计、慢查询分析
* 配置简单，UI 监控页面很强
* 常用于：企业后台项目

### 3️⃣ C3P0

* 较老
* 稳定但性能一般
* 现在很少新项目使用

### 4️⃣ DBCP / DBCP2（Apache）

* Tomcat 时代常见
* 较传统，配置多

---

## ⭐ 那“德鲁伊 Druid”到底是什么？

👉 **阿里开源的数据库连接池 + SQL 监控平台**

### 它能做 4 件关键事情：

✔️ 管理数据库连接（连接池）
✔️ SQL 执行时间统计
✔️ 查看慢查询
✔️ 可视化监控页面 👉 超好用

---

## 🖥️ Druid 监控界面示例功能

* 当前活跃连接数
* SQL 执行次数
* 慢 SQL 排名
* 每条 SQL 平均耗时
* 防止 SQL 注入（WallFilter）

非常适合：

> 后台管理系统 + 外卖系统 + Spring Boot 项目

---

## 🛠 Spring Boot 项目常见配置示例（知道即可）

```yaml
spring:
  datasource:
    druid:
      url: jdbc:mysql://localhost:3306/test
      username: root
      password: 123456
      initial-size: 5
      max-active: 20
      min-idle: 5
      max-wait: 60000
```

访问监控页面：

```
http://localhost:8080/druid
```
