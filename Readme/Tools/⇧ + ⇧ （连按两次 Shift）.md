在 **macOS 的 IntelliJ IDEA** 里，**快速查找「类 / 文件 / 符号」**，你只记住下面这 **3 个核心快捷键** 就够了 👇

---

## ⭐ 1️⃣ 查找【类名】（最常用）

**快捷键：**

```
⌘ + O
```

**用途：**

* 按 **类名** 精确/模糊查找
* 支持驼峰缩写（非常快）

**示例：**

* 输入 `UserService`
* 输入 `US` → 也能匹配 `UserService`

**适合场景：**

* 跳转到某个 Java / Kotlin / Scala 类
* 阅读源码、排查 bug

---

## ⭐ 2️⃣ 查找【文件】（不一定是类）

**快捷键：**

```
⌘ + ⇧ + O
```

**用途：**

* 查找 **任何文件**
* 包括：

  * `.xml`
  * `.yml`
  * `.properties`
  * `.md`
  * 前端文件（.vue / .ts / .js）

**示例：**

* `application.yml`
* `logback.xml`
* `pom.xml`

---

## ⭐ 3️⃣ 查找【符号：方法 / 变量 / 常量】（面试&排错神器）

**快捷键：**

```
⌘ + ⌥ + O
```

**用途：**

* 直接跳到：

  * 方法名
  * 成员变量
  * 常量
  * 枚举值

**示例：**

* `login`
* `processOrder`
* `JWT_TOKEN_NAME`

---

你说得对：**⌘ + O 在 mac 上经常是“打开(Open)”**，而且很多 App（含 Git GUI/某些工具）会占用。
在 **IntelliJ IDEA** 里，**类的模糊搜索**一般用下面这个（最稳）：

## ✅ 方案 A（最常用）：Search Everywhere（类/文件/符号都能模糊搜）

**快捷键：**

```
⇧ ⇧   （连按两次 Shift）
```

然后直接输入类名（支持驼峰/缩写），例如：

* `UserService`
* `US`
* `OrdCtrl`

> 这就是你要的“class 的模糊搜索”最通用入口。

---

## ✅ 方案 B（只搜类）：Go to Class

**默认快捷键通常是：**

```
⌘ + O
```

但你这里被占用了/冲突了，所以按了变成 Git 的功能。

### 怎么解决（IDEA 内改快捷键）

1. 打开 IDEA 设置：`⌘ + ,`
2. 搜索：`Keymap`
3. 在 Keymap 里搜索动作名：`Go to Class`
4. 右键 → **Add Keyboard Shortcut**
5. 绑定一个你喜欢且不冲突的，比如：

   * `⌘ + ⌥ + O`（很多人用）
   * `⌘ + P`（不推荐，P 常用参数提示）
   * `⌃ + ⌥ + O`（也可以）
